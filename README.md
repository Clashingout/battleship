# Battleship

A browser-based Battleship game — place your fleet, hunt the enemy on radar, and unlock special abilities when you sink their ships.

## Play online

After GitHub Pages is enabled, play here:

**https://clashingout.github.io/battleship/**

## Run locally

```bash
cd "/Users/s2406023/totally a fun game"
python3 -m http.server 8765
```

Then open [http://localhost:8765/](http://localhost:8765/)
